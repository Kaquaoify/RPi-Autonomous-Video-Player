Aide rapide · Générer le token avec rclone

Pas besoin d’installer rclone “système” sur votre appareil : il suffit de télécharger le binaire, lancer une commande, puis copier le JSON affiché et le coller dans “Token JSON” ci-dessus.

Télécharger rclone
Windows (recommandé) — étapes détaillées
  1. Téléchargez l’archive Windows sur la page “Télécharger rclone”.
  2. Dézippez le dossier (ex. sur le Bureau).
  3. Dans ce dossier, Maj + clic droit → “Ouvrir dans PowerShell ici”.
  4. Exécutez : 
    .\rclone.exe authorize "drive"
  5. Un onglet Google s’ouvre → connectez-vous et autorisez.
  6. Copiez le JSON affiché dans PowerShell, puis collez-le dans la zone Token JSON de cette page et cliquez Créer / Mettre à jour le remote.
macOS — étapes détaillées
  1. Avec Homebrew : brew install rclone puis : 
    rclone authorize "drive"
  2. Ou en portable : téléchargez l’archive macOS, dézippez, puis dans Terminal : 
    ./rclone authorize "drive"
  3. Copiez le JSON et collez-le ici, puis cliquez Créer / Mettre à jour le remote.
Linux — étapes détaillées
  1. Rapide : 
    curl -fsSL https://rclone.org/install.sh | sudo bash
  2. Puis : 
    rclone authorize "drive"
  3. Copiez le JSON et collez-le ici, puis cliquez Créer / Mettre à jour le remote.
Android (optionnel)
  1. Installez Termux → pkg update && pkg install rclone
  2. Exécutez : 
    rclone authorize "drive"
  3. Copiez le JSON et collez-le ici, puis cliquez Créer / Mettre à jour le remote.

Note sécurité : le JSON contient un refresh_token. Conservez-le privé. Vous pouvez révoquer l’accès depuis myaccount.google.com/permissions.
